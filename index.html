Alcance del Proyecto

Debido a la cantidad de problemas encontrados en la empresa, y el poco tiempo que contamos respecto a la implementación de este proyecto en particular,como alcance del proyecto, apostamos a lo siguiente:

  1. Proponer la utilización de un sistema informático para sustituir las registraciones manuales, tanto de proveedores, mercaderías, pedidos de clientes, pedidos a proveedores, compras a proveedores, tanto altas, bajas como modificaciones; de esta manera, como documentos de respaldo, solo utilizar informes impresos pertinentes para la gerencia de la organización.

  2. Mejorar la gestión por medio de un sistema informático la Gestión de compras, la gestión de proveedores, la gestión de ventas, la gestión al cliente y la gestión almacén.

  3. Capacitar a los socios gerentes y todo el personal sobre los sistemas de información computarizados a implementarse y los procesos mejorados para que todos comprendan los cambios implementados y sepan cómo trabajar de manera eficiente dentro del nuevo sistema.

  4. El software propuesto debería garantizar la seguridad de los datos del local y de los clientes, implementando medidas de seguridad como encriptación, acceso restringido y copias de seguridad regulares.

  5. El software a implementar podría permitir personalizar elementos como logotipos, colores o diseños, adaptándose a la imagen de la empresa. Además, debería ser escalable para futuras expansiones del negocio, como la apertura de sucursales adicionales.

!

Es importante destacar que el alcance del proyecto de software puede evolucionar a lo largo del tiempo debido a cambios en los requisitos prioridades o circunstancias que ocurran durante el desarrollo.
